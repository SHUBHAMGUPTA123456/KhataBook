# 📒 KhataBook

KhataBook is a simple and efficient offline accounting application built with Flutter that helps users manage customer and supplier transactions. Users can track money given and received, maintain transaction notes, and easily manage personal/business account records.

## ✨ Features

- 👤 Add and manage Customers & Suppliers
- 📞 Store contact details with phone numbers
- 💰 Record money **Given** and **Received**
- 📝 Add notes for each transaction
- 📊 Automatic balance calculation for each person
- 📂 View complete transaction history
- 📄 Generate and download PDF statements for specific users
- 📤 Share PDF statements
- 📲 Share accounting details directly through WhatsApp
- 💾 Fully Offline support
- 🗄️ Local data persistence using SQLite

---

## 🛠️ Tech Stack

- **Flutter**
- **Dart**
- **GetX**
- **SQLite**

---

## 📱 App Architecture

The application follows a clean and scalable structure:

- State Management → GetX
- Local Database → SQLite
- PDF Generation & Sharing
- Offline-first approach

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/KhataBook.git
```

Navigate to the project directory:

```bash
cd KhataBook
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
